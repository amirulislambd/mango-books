# 📚 Mango Books

An online book borrowing platform where users can explore, search, and borrow books digitally.

🔗 **Live Site:** [https://mango-books.vercel.app](https://mango-books.vercel.app)

## ✨ Features

- Browse & search books with category filtering (Story, Tech, Science)
- Borrow books digitally with a single click
- Secure authentication (Email/Password + Google) via BetterAuth
- User profile with update option
- Private routes for book details & profile
- Responsive on mobile, tablet & desktop
- Animated Swiper banner & scrolling marquee

## 🛠️ NPM Packages Used

`next` · `react` · `better-auth` · `@better-auth/mongo-adapter` · `mongodb` · `@heroui/react` · `@heroui/styles` · `tailwindcss` · `swiper` · `react-fast-marquee` · `react-hook-form` · `react-toastify` · `react-icons`

## 🚀 Run Locally

```bash
git clone https://github.com/amirulislambd/mango-books.git
cd mango-books
npm install
npm run dev
```

> Set up `.env.local` with `MONGODB_URI`, `BETTER_AUTH_SECRET`, `BETTER_AUTH_URL`, `GOOGLE_CLIENT_ID`, `GOOGLE_CLIENT_SECRET`
